INFORMACIÓN

¿Otro diálogo en JS? ¿Por qué?

Sí, otro más... Pero diseñado para no complicarte la vida.

Sabemos que existen muchas opciones, pero la mayoría de las alertas nativas como alert(), prompt() o confirm() no permiten personalización de estilos ni flujo asíncrono limpio. Dialog-JS 2.0 te permite integrar diálogos hermosos y accesibles con solo una línea de código.

Diferentes tipos para tus proyectos

Alerta Clásica

Reemplaza al aburrido alert del sistema. Perfecto para informar acciones concretas.

Entrada (Prompt)

Confirmación

DESPLIEGUE

Instalación Rápida

Integra Dialog-JS 2.0 en tu flujo de trabajo en menos de un minuto.

Añade la versión compilada directamente en tu documento HTML:

<script src="https://cdn.jsdelivr.net/npm/dialog-js-2.0/dist/dialog.min.js"></script>

Instala el paquete en tu proyecto local a través de npm:

npm install dialog-js-2.0

Importa el módulo en tus archivos JavaScript principales:

import Dialog from './node_modules/dialog-js-2.0/index.js';
API REFERENCE

Parámetros de Configuración

Personaliza hasta el último detalle pasando un objeto de opciones al inicializar.

Propiedad Tipo Predeterminado Descripción
title string 'Alerta' El título de la cabecera del cuadro del diálogo.
message string '' El cuerpo de texto o HTML que se renderiza dentro del diálogo.
theme string 'theme-dark' El tema del modal: theme-dark, theme-light, theme-glass o theme-obsidian.
okText string 'Aceptar' El texto visible del botón principal de confirmación.
cancelText string 'Cancelar' El texto visible del botón secundario para cancelar (en Confirm y Prompt).
defaultValue string '' El valor por defecto relleno en el campo de entrada (en Prompt).
